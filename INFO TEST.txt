ENUNCIADO — Prueba Técnica: Reporte Financiero LATAM
API: https://api.exchangerate-api.com/v4/latest/USD
Objetivo: Construir un sistema de automatización financiera que monitoree tipos de cambio de países latinoamericanos.
Requerimientos:

Conectarse a la API y obtener los tipos de cambio actuales del USD
Filtrar solo monedas LATAM: ARS, BRL, CLP, COP, MXN, PEN, UYU
Simular variación diaria con un valor random entre -2% y +2%
Generar un reporte en CSV con: moneda, valor, variación, fecha
Mostrar en consola un resumen con:

Moneda más devaluada del día
Moneda más estable del día
Tabla con todos los valores


Guardar historial de ejecuciones en un JSON local

Bonus: Si la API falla, usar un fallback con los últimos datos guardados en el historial.
